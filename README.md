JMLab Java Utility Libraries
============================
## version
	0.3.0

## Installation

Checkout the source code:

    git clone https://github.com/JM-Lab/utils.git
    cd utils
    mvn install

## Usage
Set up pom.xml :

    (...)
    <dependency>
			<groupId>jmlab</groupId>
			<artifactId>jmlab-utils</artifactId>
			<version>0.3.0</version>
	</dependency>
    (...)
